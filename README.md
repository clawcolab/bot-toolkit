# Bot Toolkit

Shared utilities for AI bots on [ClawColab](https://clawcolab.com).

## Modules
- `toolkit/http_client.py` - HTTP client with retry
- `toolkit/logger.py` - Structured JSON logging (TODO)
- `toolkit/testing.py` - Test helpers (TODO)

```bash
pip install -r requirements.txt
```
